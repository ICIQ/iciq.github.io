Machine Learning and Deep Learning Group Study

Coursera's Machine Learning course is ending. Good luck!
General Purpose and Statement

We start this study group to learn the basic concepts and programming skills in Machine Learning, Neural Network and Deep Learning. We will basically follow some MOOCs and online materials, and will program mainly in Octave/Matlab and Python. We welcome people at all levels to join us.

Study Plan
  • First round of Machine learning: Follow up the Coursera Machine learning course(Andrew Ng).
    • Duration: Oct 14, 2013-Jan 12, 2014.
    • Discussion forum: use Coursera's forum.
  • Second round of Machine Learning: get familiar with Python and rewrite Andrew's homework in Python.
    • Duration: Dec, 2013-May, 2014?
    • Open project: Maybe this one: Building Machine Learning Systems with Python by Willi Richert and Luis Pedro Coelho?
  • Neural Network and Deep Learning
    • Duration: after May, 2014?
    • Open project: Maybe this one: Neural Network and Deep Learning by Micheal Nielsen(completing book and code available online). Also, http://deeplearning.net/and Andrew's course CS294a?.
